문자함수 연습

        0         1         2         3
        0123456789012345678901234567890
        ABCDEFGHIJKLMNOPQRSTUVWXYZabcde
           ABC   BCABC   GHI  JKLMN.....   ABC.. OPQ

length : 객체(문자(열))의 길이

substring(a,b) : 문자열 발췌(a에서부터 b앞까지

substr()

replace(a, b) : a를 b로 치환

replaceAll(a, b) : 모든 a를 b로 치환

toUpperCase() : 모든 문자를 대문자로!

toLowerCase() : 모든 문자를 소문자로!

concat(a, b) : a와 b문자열 결합

trim() : 문자열의 앞뒤(양쪽)공백을 무시..

trimStart() : 문자열 앞쪽 공백을 무시

trimEnd() : 문자열의 뒤쪽 공백을 무시

replaceAll(' ', '') : 문자열의 모든 공백을 무시

charAt(i) : 하나의 문자만 발췌

charCodeAt(i) : i번째의 문자열의 아스키코드 표출

split() : 특벙문자를 기준으로 잘라낸다. 이때 잘라낸 값들은 배열변수에 기억된다.

indexOf() : 문자(열) 검색시에 사용 : 있으면 위치값을, 없으면 -1을 반환

lastIndexOf() : 문자(열) 검색시에 사용(뒤에서부터 검색 있으면 위치값을 없으면 -1 반환

search() : 문자(열) 검색시에 사용 : 있으면 위치값을, 없으면 -1을 반환

match() : 문자열 일치여부...있으면 문자열 그대로 반환, 없으면 공백

includes() : 문자열 포함여부...포함하고 있으면 true, 없으면 false로 반환
